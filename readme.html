Importante

Este panel Kanban no cuenta con almacenamiento permanente ni backend. Toda la información se guarda únicamente en localStorage del navegador.

Esto significa que:

  • Si borras los datos del navegador, perderás la información.
  • Si usas modo incógnito, los datos no se conservarán.
  • Si cambias de dispositivo o navegador, no verás tus tableros.

Recomendación de uso

Este proyecto está pensado para:

  • Uso personal
  • Organización temporal
  • Pruebas rápidas
  • Equipos donde no se limpian los datos del navegador con frecuencia

No está diseñado para uso crítico ni almacenamiento a largo plazo.
